# Medical-Device-System-Portfolio
Embedded firmware, system debugging, and validation projects in medical devices

This repository documents selected R&D and embedded systems work in medical devices, focusing on firmware development, system-level debugging, and production-grade validation.

## Domains
- Cardiac rhythm management (Pacemakers)
- Neurostimulation (DBS)
- Embedded firmware
- System-level test and validation
- Reliability and regression testing

## Contents
- External non-programmable pacemaker firmware
- Deep Brain Stimulation (DBS) system debugging
- Automated pacemaker and circuit test systems
- Burn-in and reliability test infrastructure

> Note: This repository contains high-level documentation only. No proprietary schematics, firmware, or confidential data are shared.
